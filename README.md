#Notas

Un programita para multiplicar

```

Opciones:
      --help     Muestra ayuda                                        [booleano]
      --version  Muestra número de versión                            [booleano]
  -b, --base     Base de la tabla multiplicar               [número] [requerido]
  -l, --listar   Lista la tabla de multiplicar       [booleano] [defecto: false]
  -h, --hasta    Hasta que numero quieres multiplicar     [número] [defecto: 10]

```
